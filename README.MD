# BookRoll PDF Exporter

Kyoto University の学習で利用されるスライド共有サービス BookRoll で公開されている教材スライドを、PDF として保存する Chrome 拡張機能です。

## 特長

- BookRoll の閲覧画面から、表示中のスライドを順番に取得して 1 つの PDF にまとめます。
- 画像化したスライドをそのまま PDF にするため、ページ順を保ったまま保存できます。
- 拡張機能のポップアップから操作できます。

## 使い方

1. この拡張機能を Chrome に読み込みます。
2. BookRoll の教材スライドを開きます。
3. 拡張機能アイコンをクリックし、「PDFとして書き出す」を押します。
4. 保存ダイアログが表示されたら、任意の場所に PDF を保存します。

## 必要な権限

- `activeTab`: 現在開いている BookRoll タブに対して操作するために使用します。
- `scripting`: ページ内に処理を注入してスライドを取得するために使用します。
- `downloads`: 生成した PDF を保存するために使用します。

## 対応サイト

- `https://bookroll.let.media.kyoto-u.ac.jp/*`

## 注意事項

- BookRoll の閲覧画面でのみ動作します。
- ページ構成が変わると、スライド取得に失敗する可能性があります。
- 本拡張機能は、公開されている教材の保存を補助する用途を想定しています。利用にあたっては、所属組織や提供元の利用規約に従ってください。
